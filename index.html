Hello, I’m Tyler.

My home is Cardiff, Wales.

Background

I’m an aspiring web designer who has an interest in how the internet works and functions and also very open to learn about the process of creating web pages. I've lived in lots of different places and have worked in lots of different jobs. I’m excited to bring my life experience to the process of building fantastic looking websites.

I started learning web development in late 2017 via treehouse where I learned the basics of how to create websites via html and css code. I also went on work experience for two weeks for a web agency during my GCSE's to learn more about designing websites and be able to create them with more advanced pieces of code (html and css) compared to that learned from school.

Goals

I would like to be able to develop my skills in web design to the point in which they become second nature to me so that I can be more productive with them.

  • HTML
  • CSS
  • JavaScript

All of these skills combined should set me up to work for a web agency where I can be able to efficiently use everything ive learned into use when developing websites in the future. As well as HTML and CSS, I would like to also learn other elements such as Javascript and PHP so that as well as working for a web designer I would also be interested in making my own sites if I ever need to do so.
